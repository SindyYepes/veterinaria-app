# 🐶 VetCare - Sistema de Gestión Veterinaria

## 📌 Descripción
VetCare es una aplicación web que permite gestionar servicios veterinarios caninos, facilitando el agendamiento de citas y la administración básica de la información.

La plataforma está diseñada con un enfoque en la experiencia del usuario (UX/UI), ofreciendo una interfaz moderna, intuitiva y responsiva que permite interactuar de forma sencilla con los servicios disponibles.

---

## 🚀 Funcionalidades
📅 Registro de citas veterinarias  
🐾 Gestión de mascotas  
🩺 Selección de servicios (vacunación, consulta, peluquería, cirugía)  
📋 Visualización de citas registradas  
❌ Eliminación de citas  
💾 Persistencia de datos mediante localStorage  
📱 Diseño responsivo

## ▶️ Ejecución del Proyecto

### 🔹 Opción 1: Abrir directamente
1. Descargar o clonar el repositorio:
```bash
git clone https://github.com/SindyYepes/veterinaria-app.git
```

2. Ingresar al proyecto:
```bash
cd veterinaria-app
```

3. Abrir el archivo:
```bash
index.html
```
(O hacer doble clic sobre el archivo)

---

### 🔹 Opción 2: Servidor local (recomendado)

#### ✔ Con Python
```bash
python -m http.server 8000
```

Abrir en el navegador:
```
http://localhost:8000
```

---

#### ✔ Con Node.js
```bash
npx serve
```

Abrir en:
```
http://localhost:3000
```

## 💾 Almacenamiento

El sistema utiliza **localStorage** para:
- Guardar las citas registradas  
- Mantener la información tras recargar la página  
- Simular persistencia de datos
